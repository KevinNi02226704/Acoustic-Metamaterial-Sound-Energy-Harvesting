# 🔊 Acoustic Metamaterial for Sound Energy Harvesting

This project investigates an **acoustic metamaterial structure designed to absorb subway noise and convert it into electrical energy**.

The device combines **locally resonant phononic crystals**, **perforated plate resonators**, and **PVDF piezoelectric membranes** to achieve both **efficient sound absorption** and **energy harvesting**.

---

# 📌 Project Overview

Noise levels in subway systems can exceed **120 dB**, creating both environmental noise pollution and wasted acoustic energy.

This project proposes an acoustic metamaterial structure that:

- targets dominant subway noise frequencies  
- absorbs sound through **local resonance and bandgap effects**  
- concentrates mechanical vibration inside the structure  
- converts the vibration into electricity using **PVDF piezoelectric membranes**

The system therefore functions as both a **noise reduction device** and an **energy harvesting system**.

---

# 🧪 Simulation

Acoustic simulations were performed in **COMSOL Multiphysics**.

The model includes:

- pressure acoustics (air domain)
- solid mechanics (metamaterial structure)
- acoustic–solid coupling
- perfectly matched layers (PML)

The simulations show strong sound pressure localization inside the metamaterial structure, indicating effective absorption.

---

# 📈 Experimental Results

The absorption coefficient was measured using a **custom-built impedance tube**.

Results show:

- absorption peaks aligned with the designed resonance frequencies  
- **>95% absorption at peak frequencies**
- **>50% absorption across a broad frequency range**
- output voltage: **2.37 ± 0.29 V**
- output power: **≈5.6 mW**
